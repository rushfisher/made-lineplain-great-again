# made-lineplain-great-again
使用Easy X（C++）构建的飞机大战陨石游戏

## 一、项目概述

| 项目名称 | [飞机大战陨石] |
|:---|:---|
| 开发平台 | Windows |
| 开发工具 | Visual Studio + EasyX图形库 |
| 编程语言 | C++11 |
| 代码规模 | 约600+行 |
| 开发周期 | [如：4周（断续开发，实际上只用了一周） / 个人业余项目] |

**项目简介**：
一款纵版射击游戏，玩家控制战机在陨石雨中生存。游戏提供多种特色模式、可调节的难度参数以及独特的“板载”技能系统。项目完整实现了游戏主循环、碰撞检测、对象管理、UI交互等核心模块。

## 二、技术架构

### 2.1 整体架构

```
┌─────────────────────────────────────────┐
│              游戏主循环                   │
│  ┌─────────────────────────────────┐   │
│  │         状态机管理               │   │
│  │  菜单 → 游戏 → 结束 → 菜单       │   │
│  └─────────────────────────────────┘   │
│  ┌─────────────────────────────────┐   │
│  │         游戏逻辑层               │   │
│  │  玩家控制 / 子弹系统 / 敌人AI    │   │
│  └─────────────────────────────────┘   │
│  ┌─────────────────────────────────┐   │
│  │         渲染层                   │   │
│  │  精灵绘制 / Mask透明混合         │   │
│  └─────────────────────────────────┘   │
│  ┌─────────────────────────────────┐   │
│  │         工具层                   │   │
│  │  滑动条控件 / 碰撞检测 / 随机数   │   │
│  └─────────────────────────────────┘   │
└─────────────────────────────────────────┘
```

### 2.2 技术栈说明

| 技术项 | 具体实现 |
|:---|:---|
| 图形渲染 | EasyX图形库，使用双缓冲（BeginBatchDraw/EndBatchDraw）实现流畅画面 |
| 透明贴图 | Mask混合模式（SRCINVERT + NOTSRCERASE）实现精灵透明效果 |
| 随机数生成 | C++11 `<random>` 库（mt19937梅森旋转算法），替代传统rand() |
| 帧率控制 | GetTickCount() 时间差计算，实现帧率无关的游戏逻辑 |
| 输入处理 | Windows消息队列（peekmessage）处理鼠标/键盘事件 |

### 游戏玩法
驾驶你的飞机，躲避陨石  

存活约16s后，机身出现红石图案，你的banzai状态刷新了，按下右键，图案消失，你可以撞击陨石，置之死地而后生，消灭大量陨石  

想要特殊玩法吗，进入设置，调整你的子弹伤害，自动射击，解放双手，体验特色模式  

败北战死，你可以看看你发射的子弹，分数，消灭的陨石  

操作不会？点击帮助，帮你回忆操作
